Librè

Librè is a resource manager, it is mainly designed and created for students to help them find resources and learning materials easier and for free. It is an application that can be used offline with complete resources, so it could be used everywhere. This is a student-friendly software, both digital multimedia encyclopedia and search engine. It will also support a Local Area Network messaging for sharing modules, lessons, and books.
